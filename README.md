# airline_passenger_satisfaction
This project demonstrates basic data analysis using Python and Pandas.
This project is based on a dataset provided by the University of Science, Vietnam National University (HCMUS), which contains detailed information about flights operated by an airline company.
The primary goal of this analysis is to explore the factors that influence customer satisfaction with the airline. By examining patterns and relationships within the data, the project aims to uncover insights that can help the business improve its services and enhance passenger experience.
# Dataset Description
- The dataset includes various attributes related to passengers and flights, such as:
- Flight details: distance, departure delay, arrival delay
- Passenger demographics: gender, age, travel type, class
- Service ratings: cleanliness, in-flight service, check-in, food & drink, comfort, and entertainment
- Target variable: satisfaction (Satisfied / Neutral or Dissatisfied)
# Structure
- Cleaning data: Python
- EDA
- Collab SQL
- Build dashboard on Tableau Public
# Expected Outcomes
- A clear understanding of what drives customer satisfaction in the airline industry.
- Actionable insights for improving service quality and operational efficiency.
- Visual dashboards and data summaries to communicate findings effectively.
